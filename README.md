# Software Construction (SEN 306)

This repository contains the coursework, lab exercises, and projects developed for the **Software Construction** course during my third year. The focus of this repository is on building scalable, maintainable, and efficient software systems.

## Learning Objectives
* Applying advanced **Design Patterns** (Creational, Structural, Behavioral).
* Implementing **Unit Testing** and Test-Driven Development (TDD).
* Managing software lifecycle using **Version Control (Git)**.
* Building robust APIs and handling concurrent processes.
